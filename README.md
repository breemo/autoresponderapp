# Auto Responder App

A dual-interface web application for managing automated responses using **Next.js**, **Supabase**, and **TailwindCSS**.

## Structure
- `/login` → Login page
- `/admin` → Admin dashboard
- `/admin/clients/add` → Add new client form
- `/client` → Client dashboard

## Setup
1. Create a Supabase project.
2. Copy `.env.local` and add your Supabase URL + Key.
3. Run locally or deploy via [Vercel](https://vercel.com).
